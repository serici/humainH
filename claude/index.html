    Démarche du projet

    L'angle d'analyse : l'universalité comme grille de lecture

    Ce projet n'a pas pour but de prouver que toutes les civilisations convergent vers la même vérité. Il utilise l'universalité comme instrument d'analyse : pour chaque époque, chaque civilisation, la question posée est double — qu'est-ce qui se retrouve partout, indépendamment ? Et qu'est-ce qui reste propre à ce contexte particulier ?

    L'enjeu n'est pas idéologique. C'est une démarche comparatiste : cartographier les ressemblances et les différences, les comprendre, et remonter aux causes.

    La question centrale

    Les idées ont-elles façonné l'histoire, ou l'histoire a-t-elle façonné les idées ?

    Les deux directions sont explorées. Les structures économiques, les contraintes géographiques, les révolutions techniques, les mutations biologiques — autant que les systèmes de pensée, les religions, les philosophies morales. Le projet cherche à identifier quand c'est la matière qui précède la conscience, et quand c'est l'inverse.

    Ce qui est évité

    L'eurocentrisme : l'histoire européenne est un fil parmi d'autres, pas la norme à laquelle les autres civilisations seraient comparées. Le corpus de références intègre intentionnellement des penseurs indiens, africains, chinois, iraniens.

    Le réductionnisme : une seule explication (géographique, économique, religieuse…) ne suffit jamais. Les convergences sont réelles mais leurs causes sont plurielles.

    Le téléologisme : l'histoire ne "progresse" pas vers un but prédéfini. Les convergences observées ne sont pas une "marche vers la raison" — ce sont des faits à expliquer, pas à célébrer.

    La Période Axiale — un exemple de convergence

    Entre ~800 et 200 av. J.-C., des mutations philosophiques similaires se produisent simultanément et indépendamment dans des civilisations sans contact prouvé :

    • Grèce : naissance de la philosophie, Socrate, Platon, Aristote, Stoïciens
    • Inde : Upanishads, Bouddha, Mahavira (Jaïnisme)
    • Chine : Confucius, Laozi, Zhuangzi
    • Proche-Orient : prophètes hébreux, Zoroastre

    Ce phénomène — le même "bond conceptuel" sans contact entre ces civilisations — est l'un des cas les plus documentés de convergence indépendante dans l'histoire humaine. Il sert ici de point de référence emblématique, pas d'axe organisateur du projet. Des convergences comparables apparaissent à d'autres moments : l'émergence des grandes religions mondiales (IVe-VIIe s.), la Renaissance humaniste, les Lumières, les révolutions industrielles simultanées.

    Granularité croissante

    Le niveau de détail augmente en se rapprochant du présent. Les périodes les plus anciennes (préhistoire, premières civilisations) sont traitées en grandes esquisses conceptuelles. Les périodes récentes permettent une analyse plus fine — idéologique, économique, scientifique, politique. Ce n'est pas un manque de rigueur pour les périodes lointaines : c'est que les sources et les angles d'analyse se multiplient à mesure que l'histoire se densifie.
